Functions

Functions are the basic "building blocks" of JavaScript. Functions allow JavaScript to interact with HTML and Web Browsers. Using functions, this allows developers to call code many times without repetition.

Functions are composed of a function declaration, parameters, and arguments. Functions must first be declared using function and function name; following next are the parenthesis (). The parenthesis () are necessary for the function to run while being the visual identifier that this code is a function. Next, parameters may be established within the parenthesis (). The function parameters input arguments to be passed to the function.

For example:

function name(parameter/argument, parameter/argument, ...) {

statement;

}

We then call or invoke the function by: function name();

Objects

Objects are containers for data values. Many different element data properties may be stored in objects. As well as many different values may be assigned to those properties.

Values for objects are coded as name:value pairs (name and value separated by a colon). Objects are containers for named values called properties or methods. The name:value pairs are denoted as key:value. The key represents the property of the object while the value represents the actual value of the property.

For example:

let user = {username: 'johnDoe', password: true, hobbies: 'javascript', age: 28, motto: 'booyeah!'}

To get the keys and values of these properties, dot notation (.name) must be used.

Here is an example of how to get a value from a property:

let user = {username: 'johnDoe', password: true, hobbies: 'javascript', age: 28, motto: 'booyeah!'}

To get the value for motto, we will use .dot notation. Here is our code:

console.log(user.motto);

//'booyeah!'

Arrays

Arrays are a special kind of object. They are used to store a value or multiple values. Such values may be 'strings', booleans, numbers, objects, and/or other arrays. Arrays are important in JavaScript. Arrays allow a simplistic avenue to get and set values.

Here is an example:

let arr = ['array', 23, true, {username: true, password: false}, [1, 2, 3, 4, 5, 6]];

To get the value(s) in an array, bracket [] notation is required. Array values are based upon a binary index; meaning the value stored in the beginning of the array starts with zero (0). The next index number is 1, 2, 3, ....

For example:

Take our above array:

let arr = ['array', 23, true, {username: true, password: false}, [1, 2, 3, 4, 5, 6]];

To get the number 23 in the arr array, we code arr[1];.

Although the number 23 is the second element in the array, it is placed at index number one (1), since we begin with index number zero (0).

Other methods exist to manipulate the values in arrays. The most common methods are:

  • .push - Allows us to add a value to the end of an existing array.
  • .pop - Allows us to extract the last element in the array and return it.
  • .shift - This extracts the first element in an array and returns it.
  • .unshift - This method adds an element to the beginning of an array.

These methods must be used to the end of the array argument. Like so:

arr.push("element to be added to the end of the array");
